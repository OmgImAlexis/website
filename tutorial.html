This tutorial will help you get Caddy running for the first time. If you're new to this kind of thing and want more guidance, read the guided tutorial.

Get Started

  1. Download Caddy and put it in your PATH.

  2. cd to your website's directory.

  3. Run caddy.

Caddy's default port is 2015, so open your browser to http://localhost:2015 to see it working. If you see a 404 error, then Caddy is working but the root of your site is missing an index file.

Congratulations! You're up and running.
